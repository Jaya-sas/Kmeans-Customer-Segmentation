# Kmeans-Customer-Segmentation
## Objective
To segment mall customers based on income and spending behavior using KMeans clustering.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Steps Performed
1. Loaded mall customer dataset
2. Selected income and spending score features
3. Applied feature scaling
4. Used Elbow method to find optimal clusters
5. Applied KMeans clustering
6. Visualized customer segments
7. Saved segmented dataset

## Results
- Optimal clusters: 5
- Customers grouped into meaningful segments for business strategy

## Output Files
- Elbow plot
- Cluster visualization
- Mall_Customers_Segmented.csv

## Conclusion
This project demonstrates unsupervised learning for real-world business use cases.
